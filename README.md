# congo-open-data-geospatial
Portail de visualisation de l'organisation adminastrive du Congo – République du Congo
Ce portail a pour objectif de faciliter l'accès, la visualisation et le téléchargement des données géospatiales officielles relatives à l'organisation administrative de la République du Congo.
Ce portail, développé par l’Institut Géographique National (IGN), vise à vulgariser l’accès aux données officielles afin de permettre au grand public (étudiants, chercheurs, professionnels, etc.) de les consulter et de les utiliser dans leurs travaux.
